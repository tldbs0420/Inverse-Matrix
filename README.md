# 사용법 : 
모드 선택 :
  1 : 역행렬 계산 및 비교 모드 (과제)
  2 : Ax = b 방정식 해 계산 및 비교 모드 (추가 구현)
  -1 : 프로그램 종료

행렬 입력 : 
  차수 먼저 양의 정수의 형태로 입력합니다.
  -1 입력시 모드 선택으로 돌아갑니다.

  차수 입력 후 띄어쓰기로 구분하여 행렬의 내용물을 입력합니다.

  2번 모드의 경우 b 벡터도 입력합니다.
